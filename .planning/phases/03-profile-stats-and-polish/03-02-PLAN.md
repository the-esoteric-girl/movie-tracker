Make the following changes to src/index.css. Only touch the :root block — do not change anything outside it.

**Step 1: Choose blue-gray hex value**
The blue-gray must complement `--bg-primary: #13121f` and sit between the surface color and a muted accent. A suitable value is `#6b7a99` — a desaturated slate-blue that reads as "secondary/inactive" against the dark purple background. Use this unless you find a better complement (document your choice in a comment).

**Step 2: Add --accent-muted token**
In the Accents section, after the three existing accent tokens, add:
```css
--accent-muted: #6b7a99; /* blue-gray — default shadows, secondary UI chrome */
```

**Step 3: Update --accent-shadow to use --accent-muted**
Change:
```css
--accent-shadow: #c084fc;
```
To:
```css
--accent-shadow: var(--accent-muted); /* default resting shadow — blue-gray */
```

**Step 4: Add --shadow-lavender variant for active/selected/focused states**
After the existing --shadow-lg line, add:
```css
/* Lavender shadow variant — use for active, selected, focused states only */
--shadow-sm-active: 2px 2px 0px var(--accent-lavender);
--shadow-md-active: 4px 4px 0px var(--accent-lavender);
--shadow-lg-active: 6px 6px 0px var(--accent-lavender);
```

**Step 5: Update semantic role comments**
Update the comment block for semantic roles to reflect the new four-color system:
```css
/* ── Semantic color roles ── */
/* --accent-lavender → primary interactive: buttons, active nav, active toggles */
/* --accent-pink     → emotion/danger: modal titles, destructive actions        */
/* --accent-lime     → confirmed/ratings: stars, seen indicators, success       */
/* --accent-muted    → default shadows, secondary borders, inactive chrome      */
--accent-shadow: var(--accent-muted);
--accent-highlight: var(--accent-lime);
--accent-interactive: var(--accent-lavender);
```

Do NOT touch any CSS outside the :root block. Do NOT change --accent-lavender, --accent-pink, --accent-lime values. Do NOT change --shadow-sm/md/lg formulas (they will automatically update because --accent-shadow now points to --accent-muted).
